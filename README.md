# ♾️ OptiWealth

> Stop aimlessly draining your wallet. Track the fun. Lock the wealth. Live completely guilt-free.

OptiWealth is an engineering-driven, client-side personal finance simulation platform designed specifically for young adults. Instead of relying on boring corporate accounting sheets or tedious line-item expense tracking, OptiWealth introduces a gamified, rule-based interactive playground paired with a real-time conversational onboarding flow to help users stop aimless capital bleeding and lock down long-term financial independence.

---

## 🚀 Key Architectural Features & Rules Applied

The engine enforces **Four Rules of Financial Survival** calculated via real-time data entry loops:

* **⚡ 30-Day Impulse Cooling Vault (Cash Velocity Guard)**
  Forces a structural, psychological 30-day delay countdown on non-essential, high-ticket luxury/tech assets before checkout authorization to kill 3 AM impulse shopping loops.

* **🧩 The Guilt-Free 50/30/20 Matrix**
  Automates paycheck division across *Needs* (50% structural cap), *Wants* (30% discretionary cap), and *Savings* (20% compounding floor) dynamically without manual entry friction.

* **🚗 The 20/4/10 Vehicle Affordability Check**
  Evaluates auto loans using empirical metrics: requiring a down payment of 20% or more, capping financing tenures to 4 years or less, and locking total recurring operational transit costs safely beneath 10% of gross monthly income.

* **🎯 The 25x True Freedom Multiplier (Inverse Freedom Number)**
  Calculates your nest-egg destination point instantly by applying the empirical **4% Safe Withdrawal Rule** against your expected monthly survival lifestyle metrics.

## 🎨 Immersive Light-Mint UX Showcase

* **Conversational Terminal:** A custom-styled, high-transparency `bg-white/35` glassmorphic slider panel centered gracefully over the background tracking vector models.
* **Cinematic Fluid Canvas Background:** Runs a responsive client-side HTML5 canvas particle simulation handling floating global currency trails.
* **State-Driven Interactive Physics:**
  * *Step 1 (Income Generation):* Generates a steady waterfall stream of currency notes and gold coins descending down the viewport layout.
  * *Step 2 (Capital Savings):* Smoothly draws an executive leather bi-fold canvas wallet on the right third of the screen, curving all falling coins elegantly inside via mathematical attraction loops.

## 🛠️ Technical Stack & Configuration

The application is engineered on top of an agile frontend web stack:

* **Framework:** React 19 / Next.js 14+ (App Router, strict TypeScript architecture)
* **Styling Engine:** Tailwind CSS 3+ (Utilizing `backdrop-blur` glassmorphism modules)
* **Typography Tokens:** Plus Jakarta Sans (Geometric Headers) / Inter (Monospaced Alphanumeric Readouts)
* **Graphics Engine:** HTML5 Canvas API backed by a localized `requestAnimationFrame` state machine loop
* **Client-Side Validation:** Debounced inline uniqueness API mocking, regex input pattern sanitizers, and Zxcvbn-style input string verification.

## 🏃 Local Development Quickstart

Ensure you have Node.js installed on your machine environment, then run the configuration suite locally:

1. **Clone the repository track:**
   ```bash
   git clone [https://github.com/your-username/optiwealth.git](https://github.com/your-username/optiwealth.git)
   cd optiwealth

```

2. **Install dependency nodes:**
```bash
npm install

```


3. **Launch the local playground development server:**
```bash
npm run dev

```


4. Open your browser framework and point it toward `http://localhost:3000` to verify the responsive full-screen canvas layout!
